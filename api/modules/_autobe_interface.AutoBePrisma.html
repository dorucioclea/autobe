Namespace AutoBePrisma

Namespace containing all interfaces for generating Prisma ORM schema through code generation.

This namespace defines the structure for converting requirements into Prisma schema files, following the patterns observed in the uploaded schema files where models are organized by business domains (Systematic, Actors, Sales, Carts, Orders, Coupons, Coins, Inquiries, Favorites, Articles).
