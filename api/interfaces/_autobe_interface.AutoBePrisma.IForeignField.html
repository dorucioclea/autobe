Interface IForeignField

Interface representing a foreign key field that establishes relationships between models.

Foreign keys create associations between models, enabling relational data modeling. They can represent 1:1, 1:N, or participate in M:N relationships through junction tables.

  • name: string & Pattern<"^[a-zA-Z_][a-zA-Z0-9_]*$">

    Name of the relation property in the Prisma model.

    Used to access the related model instance. Usually a descriptive name of the relationship. Examples: "customer", "channel", "parent", "snapshot"

  • targetModel: string

    Name of the target model being referenced.

    Must match exactly with an existing model name in the schema. Examples: "shopping_customers", "shopping_channels", "bbs_articles"
